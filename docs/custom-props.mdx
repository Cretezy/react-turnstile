# Custom props

You can pass the props listed [here](/props) into the `Turnstile` component. Here is an example:

<CodeGroup>
  ```jsx
  import { Turnstile } from '@marsidev/react-turnstile'

  export default function Widget() {
    return (
      <Turnstile
        siteKey='{{ siteKey }}'
        className='fixed bottom-4 right-4'
        options={{
          action: 'submit-form',
          theme: 'light',
          size: 'compact',
          language: 'fr',
        }}
        scriptOptions={{
          appendTo: 'body'
        }}
      />
    )
  }
  ```

  ```tsx

  }
  ```
</CodeGroup>
