# Basic usage

The only required prop is the `siteKey`. You can get your site key from the [Cloudflare page](https://developers.cloudflare.com/turnstile/get-started/).

```tsx
import { Turnstile } from '@marsidev/react-turnstile'

function Widget() {
  return <Turnstile siteKey={{ sitekey }} />
}
```
